# DocuTree

This is a program that provides a **tree view** of the directory and lists all files with **designated extensions**.

## Usage

```bash
python main.py

-i path/to/folder
-e extension
```

## Example File Structure

```markdown
# Example

.
├── file1.extension
├── subfolder

---

## file1.extension

```extension
content of file1
\```

## subfolder/file2.extension

```extension
content of file2
\```
```
